# LOS Calculator (Signalized) — Static Website

This is a simple, educational, **HCM-inspired** LOS calculator for signalized intersections.
It calculates per-lane-group delay/LOS and an intersection weighted average delay/LOS.

## Run locally (no internet, no domain)
1. Download the folder.
2. Open `index.html` in any browser.

## Deploy for free (no domain required)
### Option A: Vercel
1. Create a GitHub repo and upload these files.
2. Go to Vercel → New Project → Import the repo → Deploy.
3. You’ll get a free URL like `https://your-project.vercel.app`.

### Option B: Netlify
1. Drag-and-drop this folder into Netlify “Deploys”.
2. You’ll get a free URL like `https://your-site.netlify.app`.

### Option C: GitHub Pages
1. Create a GitHub repo.
2. Upload files.
3. Settings → Pages → Deploy from branch.
4. You’ll get `https://yourusername.github.io/repo-name/`.

## Notes
- This is an educational estimator (not official HCM software).
- The “Advanced inputs” panel includes optional factors (PHF, HV%, lane width, arrival type, etc.).
